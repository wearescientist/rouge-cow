# 肉鸽牛牛 - 游戏设计文档

## 1. 游戏概述

### 1.1 核心概念
- **类型**: Roguelike 俯视角射击
- **玩法**: 以撒的结合 + 吸血鬼幸存者
- **风格**: 可爱像素风 + 轻微恐怖元素

### 1.2 背景故事
草原上的动物们被地底钻出的寄生怪物感染，变成丧尸。小奶牛哞哞修行归来，发现父母被抓走，于是闯入地底拯救家人。

## 2. 核心玩法

### 2.1 基础操作
| 操作 | 按键 | 说明 |
|------|------|------|
| 移动 | WASD / 方向键 | 自由移动 |
| 攻击 | 自动 | 自动瞄准最近敌人 |

### 2.2 游戏循环
```
进入房间 → 击败所有敌人 → 获得经验/道具 → 进入下一房间 → Boss战 → 下一层
```

### 2.3 升级系统
- 击败敌人掉落经验宝石
- 经验满后升级，三选一获得道具
- 道具永久叠加，形成Build

## 3. 角色系统

### 3.1 基础素体（4种风格可选）
1. **简约大头** - 以撒风格，极简
2. **日系美型** - 动漫感，精致
3. **马里奥式** - 厚重，职业感
4. **Q版圆球** - 超可爱，挂件风

### 3.2 头套系统（可叠加）
- 🐮 **牛角头盔** - 变奶牛战士
- 🐷 **猪鼻面具** - 变猪猪坦克
- 🐔 **鸡冠头饰** - 变小鸡快攻
- 🦠 **寄生触须** - 变异形态

## 4. 道具系统

### 4.1 被动道具
| 图标 | 名称 | 效果 |
|------|------|------|
| 🥛 | 浓缩牛奶 | 伤害+1 |
| 🍀 | 幸运草 | 暴击率+15% |
| 🔔 | 金铃铛 | 攻速+25% |
| 🦬 | 牛角 | 攻击范围+50% |
| 🪽 | 翅膀 | 移速+20% |
| ❤️ | 大心脏 | 生命上限+1 |

### 4.2 道具叠加效果
- 多个同类道具效果叠加
- 视觉效果也会改变（如拿到3个牛角，角会变大）

## 5. 敌人设计

### 5.1 普通敌人
| 敌人 | 特点 | 攻击方式 |
|------|------|----------|
| 变异小鸡 | 速度快，血量少 | 啄击 |
| 变异小猪 | 速度慢，血量多 | 冲撞 |
| 变异小羊 | 会闪现 | 角击 |

### 5.2 Boss设计
1. **寄生母鸡** - 召唤小鸡，全屏落羽
2. **铁甲猪王** - 冲锋，地震
3. **母体** - 多阶段，召唤所有类型

## 6. 关卡设计

### 6.1 草原层（回忆）
- 地板：绿色草地
- 敌人：较弱的小动物
- Boss：被寄生的牧羊犬

### 6.2 洞穴层（主要）
- 地板：棕色洞穴
- 墙壁：岩石带紫斑
- 敌人：各种寄生怪物

### 6.3 深渊层（后期）
- 地板：紫黑色虚空
- 敌人：精英怪物
- Boss：母体

## 7. 美术规范

### 7.1 分辨率
- 角色：32x32 基础，4倍放大显示
- 道具：20x20
- UI：按需
- 瓦片：32x32

### 7.2 配色原则
- 主角：明亮温暖色
- 敌人：原色+紫色寄生元素
- 环境：暗色系+紫斑
